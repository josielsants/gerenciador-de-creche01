# 📦 Legacy - Código Anterior

Esta pasta contém o código anterior do projeto "Gerenciador de Creche" antes da refatoração para Spring Boot.

## 📋 Arquivos

- **Aluno.java** - Classe que representa um aluno
- **Aplicativo.java** - Classe principal da aplicação anterior
- **Coordenador.java** - Classe para coordenador
- **Cuidador.java** - Classe para cuidador
- **Educador.java** - Classe para educador
- **Estagiario.java** - Classe para estagiário
- **Funcionario.java** - Classe base para funcionários
- **Main.java** - Ponto de entrada anterior
- **Matricula.java** - Classe para matrícula de alunos
- **Pessoa.java** - Classe base para pessoas
- **Professor.java** - Classe para professor
- **Responsavel.java** - Classe para responsável
- **Turma.java** - Classe base para turmas
- **TurmaCreche.java** - Classe para turma creche
- **TurmaInfantil.java** - Classe para turma infantil
- **TurmaPre.java** - Classe para turma pré-escola
- **Utils.java** - Utilitários da aplicação

## 🔄 Referência

Este código pode ser consultado como referência para entender a estrutura anterior do projeto. Para o desenvolvimento novo, utilize a estrutura Spring Boot em `src/main/java`.

## 📌 Nota

A aplicação foi refatorada para usar **Spring Boot 3.2** com arquitetura em camadas (MVC), fornecendo uma base mais profissional e escalável para desenvolvimento futuro.

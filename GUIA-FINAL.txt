🚀 COMO USAR AGORA

1️⃣ No IntelliJ IDEA:
   - Abra o arquivo pom.xml
   - Clique em "Load Maven Project" ou "Maven" → "Reload Project"
   - Espere 2-3 minutos para baixar todas as dependências

2️⃣ Para rodar a aplicação:
   - Clique no botão ▶️ verde na classe GerenciadortarefasApplication
   OU
   - Terminal: mvn spring-boot:run

3️⃣ Acessar a API:
   - Swagger UI: http://localhost:8080/swagger-ui.html
   - Lista de tarefas: http://localhost:8080/api/tarefas
   - Banco H2: http://localhost:8080/h2-console (user: sa, sem senha)

4️⃣ Rodar os testes:
   - Clique botão direito em TarefaServiceTest.java → "Run Tests"
   OU
   - Terminal: mvn test

📡 PRINCIPAIS ENDPOINTS

GET    /api/tarefas              → Lista todas
GET    /api/tarefas/{id}         → Busca por ID
POST   /api/tarefas              → Cria nova
PUT    /api/tarefas/{id}         → Atualiza
DELETE /api/tarefas/{id}         → Deleta
PATCH  /api/tarefas/{id}/concluir     → Marca como concluída
PATCH  /api/tarefas/{id}/reabrir      → Reabre

🧪 TESTE RÁPIDO COM cURL

Criar tarefa:
curl -X POST http://localhost:8080/api/tarefas \
  -H "Content-Type: application/json" \
  -d '{"titulo":"Aprender Spring Boot","descricao":"Conceitos fundamentais"}'

Listar tarefas:
curl http://localhost:8080/api/tarefas

Buscar por ID:
curl http://localhost:8080/api/tarefas/1

Atualizar:
curl -X PUT http://localhost:8080/api/tarefas/1 \
  -H "Content-Type: application/json" \
  -d '{"titulo":"Spring Boot avançado","concluida":true}'

Deletar:
curl -X DELETE http://localhost:8080/api/tarefas/1
